# Getting Started

1. Add the `.env` file at the root directory, like this:
    ```properties
    FIAP_URL = https://www2.fiap.com.br
    FIAP_CLASS_NAME = CLASS_CODE
    USERNAME = USER_CODE
    PASSWORD = USER_PASS
    AWS_S3_BUCKET = BUCKET_NAME
    ```
    
2. run
    ```bash
    ./execute.sh
    ```
